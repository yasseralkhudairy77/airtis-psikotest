# AIRTIS Psikotest Webapp
Buka `app/index.html` menggunakan Live Server di VSCode.

## Struktur
- `app/assets/js/core` : modul core (session/storage)
- `app/assets/js/tests/disc` : modul DISC (items, mapping, scoring, interpretasi)
- `app/assets/js/pages` : logic per halaman
- `app/iq.html` : tes IQ / IST hasil copy dari webapp sumber
- `app/kraepelin.html` : tes Pauli Kraepelin hasil copy dari webapp sumber

## Tes Yang Tersedia
- DISC
- IQ / IST
- SPM Raven
- Pauli Kraepelin
